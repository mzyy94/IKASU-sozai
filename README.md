# Slidoon
イカすSplatoonスライドテーマ

## What's this
[実践イカパケット解析](http://www.slideshare.net/mzyy94/packetoon)で用いたスライドのテーマを一部修正したものです。
修正点は、

- インクの飛び散り絵を自作
- カラーテーマ調整
- フォントをNoto Sans CJKに統一

です。

ちなみに**スライドゥーーン**と読みます。
